Creación de rama
git branch nombreRama //creamos la rama
git branch //vericamos las ramas
git checkout nombreRama //si no estamos en ella nos pasamos a nuestra rama en la que vamos a trabajar

Subir cambios
git add . //se suben todos los cambios
git commit -m "Descripción" //Commit de mensaje descriptivo
git push origin nombreRama //Envía la rama al repositorio remoto

Ir al fork a Compare & pull request agregar descripción y dar clic en Create pull request
